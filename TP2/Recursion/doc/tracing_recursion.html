Tracing recursion¶

Tracing factorial computation¶

ap2_decorators.trace(fct)[source]¶

Decorator for tracing every call to fct. Recursive calls are indented.

Example

>>> @trace
... def fact(n):
...     if n == 0:
...         return 1
...     else:
...         return n * fact(n - 1)

>>> fact(5)
 -> fact((5,), {})
... -> fact((4,), {})
...... -> fact((3,), {})
......... -> fact((2,), {})
............ -> fact((1,), {})
............... -> fact((0,), {})
............... <- 1
............ <- 1
......... <- 2
...... <- 6
... <- 24
<- 120
120
